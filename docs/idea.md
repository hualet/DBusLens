## 定位

主要用来监控 dbus 流量，并进行事后分析的工具。 主要用来辅助分析 dbus 调用的调用热点、延时热点、错误等情况，以便从 distro maintainer  的角度进行改进。



## 功能



- cli接口：
  - 可指定 system 或者 session bus
  - 可指定 record 时长
- 全键盘操作 和 底部的操作提示

- 对外调用量 top 展示： 
  - 一级 按照  服务（进程）的调用量排序，选中点击后进入二级界面
  - 二级 按照  接口 + 具体的方法/属性/signal 排序
- 被调用量 top 展示：
  - 一级 按照  接口 +具体的方法/属性 被调用量排序，选中点击后进入二级界面
  - 二级 按照  服务（进程）的对本接口 +具体的方法/属性 调用量排序
